Object

Basically, a collection of name-value pairs

                            let object = {
                                foo: 'bar',
                                age: 42,
                                baz: {
                                    myProp: 12
                                }
                            }
                            

                            let object = new Object();
                                object.foo = 'bar';
                                object['age'] = 42;
                                let bazPropertyName = 'baz';
                                object[bazPropertyName] = { myProp: 12 };
                            }
- Object literal notation - JavaScript don't have classes.
